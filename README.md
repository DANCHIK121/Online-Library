Online Library on Python Django

Шаги для запуска сервера:
1) Скопировать этот репозиторий по ссылке https://github.com/DANCHIK121/Online-Library
2) Зайти в папку репозитория
3) Открыть эту папку в консоли
4) Ввести команду python manage.py runserver
5) Установить программу cloudpub
6) Нажать на кнопку добавить и ввести там localhost и порт 8000
7) Скопировать полученную ссылку и если перейти по ней, то откроется сайт библеотеки
